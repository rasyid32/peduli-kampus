# Frontend Placeholder

Folder ini disiapkan untuk Anggota 5.

Anggota 5 mengerjakan frontend atau integrasi tampilan di folder/frontend project yang disepakati tim. Jika tim memilih membuat frontend dalam repository ini, folder ini dapat diganti atau dikembangkan menjadi folder frontend utama.

Halaman yang direncanakan:

- Login
- Register
- Dashboard mahasiswa
- Tambah laporan
- Daftar laporan
- Detail laporan
- Dashboard teknisi/admin
- Kelola kategori
- Integrasi frontend dengan REST API
